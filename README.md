Gyt
===

Gyt is my little project to create a bare-bones git clone as a ruby gem.

### Usage - CLI

Currently, the following commands are (at least partially) implemented:

```
$ gyt init
$ gyt add ./file/path
$ gyt status
$ gyt commit -m "First commit"
$ gyt log
$ gyt checkout -b release
$ gyt checkout master
```
